Вариант 1. Транспортная сеть города
Узлы: остановки, маршруты, пересадочные узлы, районы. Рёбра: маршрут→остановка, остановка→район, пересадка между маршрутами. SHORTEST_PATH для поиска кратчайшего пути между остановками.

Вариант 2. Социальная сеть организации
Узлы: сотрудники, отделы, проекты, навыки. Рёбра: сотрудник→отдел, сотрудник→проект, сотрудник→навык, сотрудник→сотрудник (подчинение). SHORTEST_PATH для поиска связей между сотрудниками через отделы и проекты.

Вариант 3. Логистическая сеть складов
Узлы: склады, маршруты доставки, города, товары. Рёбра: склад→город, склад→товар, склад→склад (перемещение). SHORTEST_PATH для поиска пути доставки товара.

Вариант 4. Библиотечный каталог
Узлы: книги, авторы, жанры, читатели. Рёбра: книга→автор, книга→жанр, читатель→книга (взял). SHORTEST_PATH для поиска рекомендаций: книги, которые брали похожие читатели.

Вариант 5. Водоснабжение района
Узлы: насосные станции, водопроводы, дома, районы. Рёбра: станция→водопровод, водопровод→дом, дом→район. SHORTEST_PATH для поиска пути воды от станции до потребителя и выявления проблемных участков.
